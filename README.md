# RISC-V 模拟器与调试器基础框架

这是一个用于实现题目 04 的最小工程骨架，当前目标是先搭好可扩展框架，再逐步补齐 ELF、指令集、syscall 和调试器能力。

## 目录结构

- `src/`：核心源码
- `include/`：公共头文件
- `tests/`：最小测试
- `examples/`：测试程序样例预留
- `docs/`：设计文档、流程图、指令表预留

## 当前已具备

- 可编译的 C++17 工程骨架
- 内存模块最小实现
- 模拟器与调试器入口
- ELF 加载器占位接口
- 最小 smoke test

## 下一步建议

1. 补齐 RV32I 指令解码与执行
2. 接入 ELF32/ELF64 解析
3. 完成 syscall：`write/read/exit/brk`
4. 增加断点、单步、寄存器/内存查看
5. 再考虑五级流水线、F 扩展或 GDB RSP
